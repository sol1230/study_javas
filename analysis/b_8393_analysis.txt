1. 문제파악 : n이 주어졌을 때 1부터 n까지 합 구하기
             첫째 줄에 n(1<= n <= 10,000) 입력
              1부터 n까지 합 출력

2. 유추 : 1부터 n까지 합
         입력에 자연수 제한 있음
         반복문으로 얻은 합 출력

3. 주요단어 영문 이름
    입력 : input
    출력: output
    합 : sum
    자연수 : number

4. 테스트케이스
 3 --> 6
 5 --> 15
 1 --> 1

5. 프로그래밍 순서
    반복값 입력
    합 구함
    합한 값 출력
